# 檔案結構

?> 此模板是 `Vuetify` 為基礎建構下開發，檔案結構如下。

```text

doc
 |
 ├── public /
 |      └── ...
 ├── src /
 │      ├── assets/
 │      │   └── images/               // 檔案圖片放置處
 │      ├── components/               // 元件資料夾
 │      │   ├── chart/
 │      │   └── ...
 │      ├── plugins/                  // 外掛資料夾（Vuetify 基本設定）
 │      │   ├── index.js
 |      |   ├── vuetify.js
 │      │   └── webfontloader.js
 │      ├── router/                    // 路由設定
 │      │   └── index.js
 │      ├── sass/                      // CSS設定
 │      │   └── ...
 │      ├── views/                     // 頁面
 |      |   ├── admin/
 │      │   │    └── ...
 │      │   └── layout/
 │      │        └── ...
 │      ├── App.vue
 │      └── main.js                    // 頁面設定
 ├── index.html
 ├── package.json
 ├── README.md
 └── vite.config.js


```
